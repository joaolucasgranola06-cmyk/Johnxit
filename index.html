JohnXit - Xit via IP

Este site foi criado para carregar o xit diretamente no navegador usando seu IP.

Passos

  1. Execute o servidor: npm run serve
  2. Abra o navegador no dispositivo ou em outro aparelho da mesma rede
  3. Use o endereço abaixo para acessar o site
  4. Toque em START Xit para ativar o aimbot

Endereço de acesso

Cole este endereço no navegador do dispositivo que vai usar o xit.
