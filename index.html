Este site serve para carregar o código JavaScript do aimbot via browser e permitir uso por IP.

Abra esta página no dispositivo ou navegador que irá acessar o script:

http://localhost:8080

Use o menu do navegador para abrir o console e execute:

AndroidAimbotBridge.startAimbot()

Botão útil para testar se a ponte está disponível:

Status: aguardando...
